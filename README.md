## ðŸ› ï¸ Getting Started

### Prerequisites
- Node.js (v18+)
- NPM or Yarn
- A Google Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/nebula-builder.git
   cd nebula-builder
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment**
   Create a `.env` file in the root directory:
   ```env
   API_KEY=your_google_gemini_api_key
   ```

4. **Run the development server**
   ```bash
   npm start
   ```

## ðŸ“– Usage Guide

### 1. Setup & Identity
Define your project name, brand colors, and typography. Choose your target platform (Web or Mobile).

### 2. AI or Template
- **AI Magic**: Type "A fitness app for tracking gym progress" and watch the system generate screens.
- **Manual**: Start with a blank canvas or choose a pre-made template.

### 3. Flow Builder
- Click **+ Add Screen** to expand your app.
- Drag the **Cyan Dot** from one screen to another to create navigation links.
- Click the center of a card to enter the **UI Editor**.

### 4. UI Editor
- Drag components from the **Left Palette**.
- Use **Groups** to organize content side-by-side (Row) or vertically (Column).
- Configure properties (Labels, Actions, Validation) on the **Right Panel**.

### 5. Export
- Click **Export Code** to get the full source code.
- Click **Preview** to interact with your app in a simulator.
